# Rundex 버전 기록

## v0.2.0 (2026-06-14)
달릴수록 채워지는 러닝 도감 — 소셜·통계·안정성 대규모 고도화.

### 피드 / 소셜
- 인스타형 피드 카드(좋아요·댓글·🔖북마크 액션행), 댓글 바텀시트(아바타·이모지 퀵바)
- 댓글 좋아요(영구), 알림 🔔(좋아요·댓글·팔로우) + 안읽음 뱃지
- 공개 프로필(통계·공개러닝·팔로우), 팔로워/팔로잉 목록
- 피드/타인 경로 북마크 저장

### 러닝
- 시작 화면 내 위치 중앙 포커스, 추천 코스 스와이프 카드
- 🎯 목표 설정(거리/시간) + 인런 진행바 + 달성 표시
- 인런 1km 구간 랩 플래시(진동), 카운트다운 3·2·1·GO
- 결과화면 서버 단일원천(새 동·승급·칭호), 업로드 재시도 큐

### 계획
- 지도 공간 확대, 빈 지도 가이드 힌트, 코스 저장/따라뛰기/공유

### 도감
- 인벤토리 대시보드(등급별 보유·시도별 정복), 동네 클릭 상세
- 정확 행정동 경계 재임포트(전국 3246동), 테마 도감 카드

### 프로필
- Threads형(게시물/팔로워/팔로잉, ☰메뉴, 게시물·러닝기록 탭, 게시물 등록)
- 러닝 인사이트(주간 그래프·최고기록), 칭호 보관함, 코스 보관함, 한 줄 소개

### 크루
- 추천/내 크루 카드, 채팅, 챌린지 + 기여도 리더보드 + 응원 문구

### 안정성
- 익명 토큰 만료 자동 갱신(401 재시도), 미업로드 러닝 어느 화면이든 재전송
- 설정 화면(계정·러닝·정보)
